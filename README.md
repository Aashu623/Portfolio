## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/showcase-sk-dev.git
   cd showcase-sk-dev
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:8080`

## ğŸ“¦ Build & Deploy

### Local Build
```bash
npm run build
```

### Deploy to GitHub Pages

#### Option 1: Manual Deployment
```bash
npm run deploy
```

#### Option 2: Automatic Deployment (Recommended)
1. Push your code to the `main` branch
2. GitHub Actions will automatically build and deploy to GitHub Pages
3. Your site will be available at `https://your-username.github.io/showcase-sk-dev/`

## âš™ï¸ Configuration

### GitHub Pages Setup
1. Go to your repository settings
2. Navigate to "Pages" section
3. Set source to "GitHub Actions"

### Custom Domain (Optional)
1. Add your custom domain in repository settings
2. Update the `base` path in `vite.config.ts`
3. Add CNAME file in the `public` folder
